# Exercícios do Curso de Linguagem C

Este repositório contém os exercícios desenvolvidos ao longo do curso de Linguagem C, com o objetivo de praticar e consolidar os conceitos fundamentais da programação em C.

Os códigos estão organizados por tópicos, acompanhando a evolução do aprendizado, desde os primeiros contatos com a linguagem (como *Hello World* e entrada e saída de dados) até conceitos como tipos de dados, operadores lógicos e estruturas condicionais.

Este repositório faz parte do meu processo de estudo e aprendizado contínuo em programação, servindo como base de prática e consulta durante minha jornada na área de desenvolvimento.
